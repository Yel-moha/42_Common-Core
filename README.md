[Entra nella cartella: `fractol`](fractol)

Anteprima Mandelbrot:

![Mandelbrot Preview](fractol/Mandelbrot_fractal.gif)

- Render in tempo reale di frattali con MiniLibX
- Supporta: `mandelbrot`, `julia <re> <im>`
- Build: `make` dentro `fractol`
- Dipendenze interne: [`ft_printf`](fractol/ft_printf/README.md), `libft`, `minilibx-linux`

Esegui velocemente su Linux:

```bash
cd fractol
make
./fractol mandelbrot
```

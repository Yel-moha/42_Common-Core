## 📂 Repository Structure

Each project follows this structure:

```
project-name/
├── README.md           # Project-specific documentation
├── Makefile            # Build configuration
├── src/                # Source files
├── includes/           # Header files
├── libft/              # Local libft (if needed)
└── [project-specific]
```

## 🛠 Getting Started

### Clone a specific project:

```bash
git clone https://github.com/Yel-moha/project-name.git
cd project-name
make
```

### Example - Building push_swap:

```bash
git clone https://github.com/Yel-moha/push_swap.git
cd push_swap
make
./push_swap 3 2 1
```
